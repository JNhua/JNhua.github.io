# 配置相关

## <a name="虚拟机网络编辑器">虚拟机网络编辑器</a>

有三个VMnet：0，1，8 。其中VMnet0在编辑模式下才可以看到。

|  名称  |  模式  |
| :----: | :----: |
| VMnet0 |  桥接  |
| VMnet1 | 仅主机 |
| VMnet8 |  NAT   |

分别对应三种模式的网络设置。

除了默认的三个虚拟网络，还可以自己添加。

## 物理机网络适配器

对VMnetx 进行ip地址的配置。

## 虚拟机网络适配器

选择以何种网络模式上网。默认的，每种模式对应与[虚拟机网络编辑器](#虚拟机网络编辑器)的虚拟网络设置。
